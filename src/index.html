Про меня

Меня зовут Василий

Привет! Я Веб-разработчик из города Санкт-Петербург. Мне нравиться моя работа, создавать что-то новое и решать интересные задачи.
Занимаюсь Frontend-разработкой/WEB-программированием/HTML-вёерсткой.
У меня полностью кроссбраузерная, адаптивная и валидная верстка. Для обеспечения выполнения требований, предъявляемых к верстке сайта в процессе работы над проектом я использую: сборщик проектов Gulp (Webpack) для автоматизации процесса верстки, препроцессор SCSS + PostCSS для ускоренной стилизации сайта и более гибкого кода, Pixel Perfect для верстки, совпадающей с предоставленным макетом «пиксель в пиксель», БЭМ, Flexbox, Grid Layout и media-запросы для адаптивной верстки. Динамика сайта организуется собственным JavaScript кодом или уже готовыми решениями. В конце работы осуществляется проверка отображения верстки в различных браузерах на разных устройствах, код проверяется валидатором на соответствие стандартам и устраняются выявленные ошибки. Часто удается достичь оценки 95+ по PageSpeed. Увлекаюсь активным спортом, шахматами, немного лошадьми.
